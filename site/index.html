How it works

Compression Analysis

AI text is predictable and compresses better. Cross-compress against AI and human preludes using LZMA + zlib to detect statistical patterns.

Burstiness Analysis

Humans vary naturally — short sentences mixed with long, simple with complex. AI is uniformly smooth. We measure the variance.

Ensemble Scoring

Confidence-aware dynamic weighting. Strong signals dominate. Weak signals only matter as tiebreakers. Better than fixed averaging.

Sentence-Level

Not just "this doc is AI." See exactly which sentences triggered detection and at what confidence. Useful for mixed-authorship text.

CI Integration

Set a threshold, get an exit code. JSON output for parsing. Drop it in your pipeline to catch AI-generated PRs, docs, or submissions.

Zero Dependencies*

Core analyzers use Python stdlib only (lzma, zlib). Rich + Typer for CLI. ML models optional. *The asterisk is honest.
